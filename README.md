# Aura

A personal finance app for the Norwegian market. Chat with an AI assistant about your finances, track spending, manage bills, and connect bank accounts.

## Tech Stack

- **Next.js 15** — App Router, React Server Components
- **Supabase** — Auth, Postgres database, Row Level Security
- **Claude API** — AI-powered financial assistant
- **Neonomics** — Open banking integration (PSD2)
- **Tailwind CSS + shadcn/ui** — Styling and components

## Getting Started

```bash
npm install
npm run dev
```

Requires a `.env.local` with Supabase, Anthropic, and Neonomics credentials.
